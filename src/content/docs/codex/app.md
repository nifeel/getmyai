## Codex App

如果你同时提供桌面端和 CLI，文档里应该强调两者配置关系。对多数用户来说，桌面端最重要的是“能不能直接复用现有 CLI 配置”。

## App 下载

把下载入口、系统要求和版本说明写清楚，不要让用户在首页里找。

## 配置方式

参考站点的做法，桌面端复用 CLI 配置是最省心的路径。只要前面 CLI 已经接好：

- 不需要重复填一遍 Key
- 不需要重新选一次模型
- 只要保证配置目录可读即可

## 推荐补充

这类页面最好额外写上：

- 支持的平台
- 首次启动排查
- 与 CLI 的配置同步策略
