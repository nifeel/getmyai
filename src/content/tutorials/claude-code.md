## 为什么要单独写工具教程

工具接入和普通 API 调用不同，用户更关心模型可用性、流式表现和响应稳定性。

## 首页只讲结论

首页写清楚支持哪些工具即可，具体配置步骤放在教程页。

## 教程页应该包含什么

- 环境变量示例
- 模型推荐
- 失败排查
- 限流和配额建议
